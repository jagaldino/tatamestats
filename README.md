# TatameStats

Estrutura inicial do projeto usando React + Vite + TypeScript, TailwindCSS, React Router e Zustand.

## Comandos de instalacao

```bash
npm install
```

## Comandos de desenvolvimento

```bash
npm run dev
npm run typecheck
npm run build
npm run preview
```
